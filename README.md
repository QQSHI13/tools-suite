# Tools Suite

A collection of useful web tools, all in one repo.

## Tools Included

| Tool | Description |
|------|-------------|
| **API Tester** | Test HTTP requests with all methods, custom headers, and request history |
| **JSON Viewer** | Visualize, validate, and format JSON with collapsible tree view |
| **Regex Tester** | Test regular expressions with live matching and pattern library |
| **JWT Decoder** | Decode and inspect JWT tokens with expiry detection |
| **CSV to JSON** | Convert CSV files to JSON with smart parsing options |
| **Diff Viewer** | Compare text side-by-side or inline with change highlighting |
| **Keycode Logger** | Log and display keyboard keycodes and events |

## Usage

Open `index.html` in your browser to see the tools dashboard, or navigate directly to each tool's subdirectory.
