# 🔍 Regex Tester

Live regex pattern matching with syntax highlighting, substitution preview, and common patterns library.

**Live:** https://qqshi13.github.io/regex-tester/

## Features

- ⚡ **Live Matching** - See matches as you type
- 🎯 **Match Highlighting** - Visual highlights in test text
- 📚 **Common Patterns** - 12 built-in patterns (email, URL, phone, etc.)
- 🔧 **Substitution Preview** - Test replace operations in real-time
- 📋 **Capture Groups** - Click matches to see groups
- 🚩 **Flags Support** - g (global), i (ignore case), m (multiline)
- ❌ **Error Messages** - Clear validation errors

## Tech Stack

- **Vanilla JavaScript** - No dependencies
- **CSS Variables** - GitHub-style dark theme
- **GitHub Pages** - Free hosting

## Common Patterns Included

- Email addresses
- URLs (HTTP/HTTPS)
- IP addresses
- Phone numbers
- Dates (YYYY-MM-DD)
- Hex colors
- Credit cards
- HTML tags
- And more...

## Usage

1. Enter regex pattern between `/` and `/`
2. Add flags (g, i, m) in the flags box
3. Type or paste test text
4. See matches highlighted in real-time
5. Click matches to see capture groups
6. Try substitution with `$1`, `$&`, etc.
