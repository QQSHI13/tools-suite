☄️ M5Capsule Sync

Sync settings with your Pomodoro Timer

Not connected. Double-click button during 5s boot window to enter sync mode.
Requires Chrome/Edge with Web Serial API support.
LED Colors:
Red = Work
Green = Short Break
Blue = Long Break
Orange = Sync Mode
White = Boot mode (5s window for sync)
How to sync:
  1. Double-click the button during the 5s white LED boot window
  2. Device enters SYNC mode (blue LED)
  3. Click "Connect" above (requires Chrome/Edge)
  4. Adjust settings and click "Send Settings"
  5. Click "Done" when finished to exit sync mode

Protocol: PING → PONG → [SYNC/GET] → PONG (exit)

Commands: PING, PONG, GET, SYNC:key=val,..., RESET
