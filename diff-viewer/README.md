# Diff Viewer

A clean, browser-based diff tool for comparing text/code.

## Live Demo
🚀 **Live**: https://qqshi13.github.io/diff-viewer/

## Features
- Side-by-side or inline diff view
- Line numbers
- Added/removed line highlighting
- Copy diff output
- Export diff as patch file
- Shareable URLs (base64 encoded inputs)
- Word-level diff option
- Ignore whitespace option
- Virtualization for large files (10k+ lines)
- Dark theme (GitHub-style)
- Keyboard accessible
- Screen reader friendly

## Tech Stack
- Vanilla JavaScript
- diff-match-patch library for diff generation
- GitHub Pages for hosting

## Local Development
```bash
# Serve locally
python -m http.server 8000
# or
npx serve
```
