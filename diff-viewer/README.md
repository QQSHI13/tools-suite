# Diff Viewer

A clean, browser-based diff tool for comparing text/code.

## Live Demo
🚀 **Live**: https://qqshi13.github.io/diff-viewer/

## Features
- Side-by-side or inline diff view
- Syntax highlighting for code
- Line numbers
- Added/removed line highlighting
- Copy diff output
- Shareable URLs (base64 encoded inputs)
- Dark theme (GitHub-style)

## Tech Stack
- Vanilla JavaScript
- diff-match-patch library for diff generation
- highlight.js for syntax highlighting
- GitHub Pages for hosting

## Local Development
```bash
# Serve locally
python -m http.server 8000
# or
npx serve
```
