# 🌳 JSON Visualizer

Interactive JSON viewer with collapsible tree view, live validation, and format/minify tools.

**Live:** https://qqshi13.github.io/json-viewer/

## Features

- 🌳 **Collapsible Tree View** - Click to expand/collapse nested objects
- 🎨 **Syntax Highlighting** - Different colors for strings, numbers, booleans, null, keys
- ✅ **Live Validation** - Error highlighting with detailed messages
- 📊 **Stats Panel** - File size, node count, and nesting depth
- 🛠️ **Format/Minify** - One-click prettify or compact
- 📋 **Copy to Clipboard** - Quick copy formatted JSON
- 💾 **Sample Data** - Pre-loaded example to get started

## Tech Stack

- **Vanilla JavaScript** - No dependencies
- **CSS Variables** - Easy theming
- **GitHub Pages** - Free hosting

## Usage

1. Paste JSON in the left panel
2. Tree view auto-updates on the right
3. Click arrows to collapse/expand nodes
4. Use toolbar buttons for format/minify/copy

## Keyboard Shortcuts

None needed - it's all real-time!
