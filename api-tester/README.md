# 🧪 API Tester

A lightweight HTTP client that runs entirely in your browser. Test APIs without installing anything.

**Live:** https://qqshi13.github.io/api-tester/

## Features

- 🌐 **All HTTP methods** - GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS
- 💾 **Request history** - Auto-saved in localStorage
- 📝 **Custom headers** - Full header editor
- 📦 **JSON tools** - Format and minify JSON bodies
- 🔐 **Basic auth** - Built-in authentication support
- 🔍 **Response headers** - View all returned headers
- 🔗 **Shareable URLs** - Encode requests in the URL for sharing
- 🌙 **Dark theme** - GitHub-style dark mode

## Tech Stack

- **Vanilla JavaScript** - No frameworks
- **localStorage** - Persist history locally
- **GitHub Pages** - Free static hosting

## Privacy

All requests are made directly from your browser. No data is sent to any server—your API keys and request data stay on your machine.

## Usage

1. Select HTTP method
2. Enter URL
3. Add headers/body as needed
4. Click Send
5. History auto-saves for quick re-runs

## URL Sharing

Click "Copy Link" to generate a shareable URL containing your entire request configuration. Anyone opening the link will see the same request ready to send.
